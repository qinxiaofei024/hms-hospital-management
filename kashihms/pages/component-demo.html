医疗质量组件库演示

展示新一代医疗质量监测系统的组件功能

统计卡片组件 (StatCard)
统计卡片组件用于显示关键指标数据，支持图标、趋势显示和点击交互。
// 创建统计卡片示例代码 const statCard = medicalQualityComponents.createStatCard(container, { title: '患者满意度', value: '95.8%', icon: 'fas fa-heart', trend: '+2.3%', trendType: 'positive', clickable: true });
数据表格组件 (DataTable)
数据表格组件提供排序、搜索、分页和行选择功能，适用于各种数据展示场景。
// 创建数据表格示例代码 const dataTable = medicalQualityComponents.createDataTable(container, { columns: [ { key: 'name', title: '患者姓名', sortable: true }, { key: 'department', title: '科室', sortable: true }, { key: 'satisfaction', title: '满意度', sortable: true } ], data: patientData, searchable: true, pagination: true, selectable: true });
医疗图表组件 (MedicalChart)
医疗图表组件基于ECharts，提供多种图表类型和医疗数据可视化功能。
// 创建医疗图表示例代码 const chart = medicalQualityComponents.createMedicalChart(container, { type: 'line', title: '患者满意度趋势', data: { categories: ['1月', '2月', '3月', '4月', '5月', '6月'], series: [{ name: '满意度', data: [92, 94, 96, 95, 97, 98] }] }, responsive: true });
质量指标组件 (QualityIndicator)
质量指标组件用于显示关键绩效指标，支持进度条、状态显示和阈值警告。
// 创建质量指标示例代码 const indicator = medicalQualityComponents.createQualityIndicator(container, { title: '手术成功率', value: 98.5, target: 95, unit: '%', status: 'excellent', showProgress: true, thresholds: { excellent: 95, good: 90, warning: 85 } });
通知组件 (Notification)
通知组件用于显示系统消息、警告和成功提示，支持多种类型和自动消失。
// 显示通知示例代码 medicalQualityComponents.showNotification({ type: 'success', title: '操作成功', message: '患者数据已成功更新', duration: 3000, position: 'top-right' });
